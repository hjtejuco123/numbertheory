# Number Theory

## What is Number Theory?

Number Theory is a branch of mathematics that focuses on integers (whole
numbers) and their properties.

It studies: - Prime and composite numbers - Divisibility - Remainders
(modular arithmetic) - Patterns in numbers

------------------------------------------------------------------------
## Applications of Number Theory

### 1. Cryptography (Security)

Used in passwords, banking, and encryption (e.g., RSA algorithm).

### 2. Computer Science

Used in algorithms, hashing, and data structures.

### 3. Coding Theory

Used in QR codes, barcodes, and error detection.

### 4. Game Development

Used in random number generation and simulations.

### 5. Problem Solving

Improves logical thinking and proof techniques.

### 6. Combinatorics

Used in probability, counting, and decision-making.

------------------------------------------------------------------------

## Simple Examples

-   7 is a prime number
-   10 mod 3 = 1
-   GCD(12, 18) = 6
